# Web アプリ共通技術設計書

## 1. 概要

本ドキュメントは、ブラウザ完結型の Next.js Web アプリを構築するための共通技術設計である。

アプリ固有のドメインロジック、画像処理、生成仕様は、各プロジェクトの技術設計書で別途定義する。

## 2. 技術スタック

採用:

- Next.js
- React
- TypeScript
- Tailwind CSS
- npm

ディレクトリ:

```text
frontend/
docs/
```

## 3. フロントエンド責務

- 画面表示
- 入力フォーム
- クライアント側バリデーション
- ブラウザ API を利用した処理
- ローディング、エラー、成功状態の表示
- Blob URL によるプレビューとダウンロード

## 4. 開発環境

開発サーバー:

```bash
npm run dev -- -p 4000
```

アプリ固有の処理はブラウザ内で完結させ、入力データをアプリサーバーへ送信しない。

## 5. テスト・確認方針

- ESLint による静的チェック
- TypeScript による型チェック
- Next.js production build
- ブラウザでの主要操作確認
